## Usage

```smalltalk
(PrSketch
	setup: [ :aCanvas | 
		aCanvas
			size: 500 @ 500;
			background: Color veryVeryLightGray ]
	run: [ :aCanvas :frameNumber | 
		aCanvas
			fill: Color blue;
			color: Color green;
			rectangleAt: 10 @ 10 withSize: 100 @ 100;
			color: Color red;
			lineFrom: 110 @ 110 to: 300 @ 300;
			fill: Color orange;
			color: Color white;
			circleAt: 350 @ 350 withRadius: 100 ])
	frameRate: 0
```
